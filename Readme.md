This is a File Organizer tool I created using Watchdog to monitor files and organize them accordingly.
This is my first project starting as a beginner in python, so the code may be  unoptimized in some cases.

This was written in python 3.14 so install it on ur computer

This was made for windows, as i don't have that much knowledge of different operating systems, will optimise for linux later.Although I don't think there should be any problems, so feel free to give it a try.

You can edit the config file according to your needs to match the amount of folders, and the extensions you want to filter into said folders.


AND DONT FORGET to edit the config file and set your folder to be monitored accordingly.

To start the program, open the python file and execute it. 

This is an early learning project and may still have rough edges. Feedback and improvements are welcome.

Installation:
1. CLone the Repo :
    git clone <your-repo-url>
    cd file-organizer
2. Create virtual Enviorment:
    python -m venv .venv
    .\.venv\Scripts\activate
3. Install dependencies: pip install watchdog

4. Run the program by: python File Organiser.py

Stop with ctrl+c in the Terminal associated with the program.
